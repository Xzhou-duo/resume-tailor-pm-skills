# resume-tailor-pm

A Claude skill for tailoring resumes to specific job descriptions, built for **互联网产品经理（Internet Product Manager）** roles.

---

## 这个 Skill 能做什么

上传你的简历（PDF）和目标岗位 JD，Claude 会：

1. **针对 JD 改写 bullet points** — 突出最相关的经历，重新排列优先级
2. **遵循 STAR 法则** — 每条改写都有背景、行动、结果的完整逻辑
3. **个人项目/未落地项目** — 没有真实数据时，用业务思维和决策逻辑替代，体现产品 sense
4. **不编造任何内容** — 只重新包装已有经历，模糊处会先提问再修改
5. **HR 视角诚实反馈** — 改完后给出不讨好、不谄媚的真实评价，指出优势和短板

---

## 使用方式

在 Claude 对话中上传此 skill，然后：

- 上传你的简历（PDF 格式）
- 粘贴或上传目标岗位 JD
- Claude 会自动按流程处理

## 适用场景

- 应届生/转行者投递互联网产品经理岗位
- 有个人项目但缺少真实数据，需要体现业务思维
- 针对不同公司/岗位需要多个定制版本简历
